# suivi-interventions-batiments-powerbi

## Dashboard Power BI – Suivi des interventions techniques dans les bâtiments

### Objectif du projet

Ce projet a pour objectif de concevoir un tableau de bord interactif permettant de suivre les interventions techniques réalisées dans différents bâtiments, afin d’améliorer le pilotage opérationnel, le suivi des coûts et les délais de traitement.

### Données utilisées

Le dataset contient les informations suivantes :

* Type d’intervention (Maintenance, Réparation, Urgence)
* Statut des interventions (Terminé, En cours, Annulé)
* Nom des bâtiments
* Coût des interventions
* Délais de traitement
* Dates des interventions
* Répartition mensuelle des opérations

### Préparation des données (Power Query)

* Nettoyage et formatage des données
* Gestion des types de données
* Création de colonnes calculées :

  * Délai moyen de traitement
  * Nombre d’interventions
  * Coût moyen par intervention
  * Segmentation par type d’intervention
  * Analyse par bâtiment et par statut

### Outils utilisés

* Power BI
* Power Query
* DAX
* Excel / CSV
